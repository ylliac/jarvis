Contact

Please first check the forums on the CloudGarden website for the latest information, and read the suggestions on the "Online Support" page.

Support for the evaluation and commercial versions of the software is available by contacting jsapi@cloudgarden.com.

If submitting a bug report, please make it as detailed as possible and include at least the following details:

  • A runnable code sample to illustrate the bug.
  • Which of the supplied examples work and which don't
  • What OS and JDK you are using.
  • What speech engines you have installed and are using.
  • What development environment you are using - eg, Notepad, Netbeans, JBuilder...

    • If an example is crashing, a full copy of the output dumped by the example.


Consultation
Consultation can be offered in the following areas:
  • Java Speech and it's use in web, network and desktop applications
  • Java Speech and Swing
  • Java Speech and database applications (JDBC)
  • Java Speech and RTP
Please contact jsapi@cloudgarden.com for more information.
