Requirements

Software
  • Sun's JDK/JRE 1.4, 1.3, 1.2 or 1.1 -  NOTE: Although the Synthesizers work well with all versions, some problems have been observed using Recognizers with JDK 1.2.2 (but not JDK 1.3 or 1.4) which seem to be Swing-related, so it is recommended that Java 1.3 or better be used for applications using Recognizers.
  • Sun's Java Media Framework 2.1.1 - In order to use all the IO features of the com.cloudgarden.audio package, Sun's JMF 2.1.1 or better must be installed. However, if the getDataSink and getDataSource methods of CGAudioManager are not used, then only the javax.sound package is required, and this is an integral part of the Java 1.3 implementation - if Java 1.1 or 1.2 is used you should install the full JMF 2.1.1
  • One or more SAPI 5.1, 5.0 or 4.0 compliant speech engine(s). Several are described here - many are free for download.
  • Microsoft Windows 98, Me , NT4, 2000 or XP operating system.
  • Sun's JSAPI specification for documentation of classes and methods, as well as JSGF and JSML specifications.

    •  
Hardware

On a 400MHz PentiumII system with 256MB RAM, performance is good.
On a 133MHz PentiumII with 64MB performance is still satisfactory for command grammars, but rather slow for dictation.

No special sound card is required - most should work.
